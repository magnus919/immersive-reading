# Copyright And Attribution

Use this reference before generating full-text reading editions.

## Rights Gate

Ask whether the user has permission to reproduce the full source text. Good answers include:

- the user wrote it
- the source is public domain
- the source has a compatible license
- the user has explicit permission
- the output is private/internal and the user accepts responsibility

If rights are unclear, do not mirror the whole work. Build an excerpt-based study guide instead.

## Allowed Safer Outputs Without Reproduction Rights

- summary-only chapters
- short excerpts for commentary
- user-provided notes
- reading guide with links to the original
- section titles and original-source pointers

## Attribution Requirements

Every generated article must include:

- original author, if known
- source URL or file/source description
- clear statement that the generated site is independent
- original link in the hero or contents utilities when available

Avoid wording that implies ownership of the original source.
